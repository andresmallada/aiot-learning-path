# AIoT Learning Path

Una colección curada de rutas de aprendizaje y guías sobre IoT e IA en los principales proveedores cloud.

> **Disponible actualmente:** Microsoft Azure IoT

## ☁️ Microsoft Azure IoT

Una guía estructurada por el ecosistema IoT de Microsoft Azure, organizada desde los conceptos fundamentales hasta los servicios más avanzados. Ideal para quienes están dando sus primeros pasos.
